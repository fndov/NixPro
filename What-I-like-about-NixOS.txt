NixOS as it stands currently has so much to appreciate that I often cannot recall it all.
1. The most packages.
* It does not matter if they're closed source or open source, a package will never be rejected for a stupid reason like steam not being in GuixOS.
2. Only on NixOS can you mix stable and unstable packages.
* I have my desktop stable and all of my apps and commands unstable, something I've always dreamed of and wanted since using linux.
3. "NixOS never breaks". I do not have a thumbdrive with an ISO on it at all and haven't for a long time, three years at this point, because I have never had NixOS break. I have never used prior generations other than explicity when trying out random kernel configs. Unlike with Archlinux or any other generic Linux OS where nobody has used their OS for more than a month; and unfortunatly extreamly often people are reinstalling because they don't know how to fix something.
4. Built in command to create a bootale ISO clone of your system since 24.05 or around there. No other system has this.
5. NixOS comes with a graphical installer. Many, many LinuxOSes do not have this and don't think its a problem when it objectively is.
6. The package pipeline with the NixPKGS github and Hydra build system make perfect since, I've asked many Gentoo users where they get their packages from and they have no idea, I've even emailed the gentoo teams and gotten rough answers; but still no other package manager compares.
7. NixOS with its atomic updates is one of the only Linux systems I trust with enabling automatic upgrades, and automatic collection of prior generations; when its the only OS that implements it on a real level.
8. NixOS has the best rollback system I have ever seen, other things like 'TimeMachine' which directly clone your entire os and then MV it to overwrite your current system when you want to go back is the most **braindead** thing I have ever seen. BTRFS Snapshots don't count and are not distro exclusive so you cannot claim that 'your OS' has an exclusive rollback system, it applies to everyone so it's not a valid point.
9. NixOS has the most blogs on it than any other Linux Distro, because people love it.
10 People say that "nixos is super hard to learn" when it is actually the easiest. Too many people come into it expecting it to be like any other generic linux distro, when it is not. The best thing to do objectively is forget everything you think you know about how linux works.
* I objectively couldn't use linux or feel at all confident in the terminal if I had to make changes manually, with nixos I open the packages and options website and paste code and close the terminal and the system will upgrade by itself later, compare this to any other distro when installing KMSCON the kernel module that replaces the TTY and gives you a fully color and gpu supported terminal, on ArchLinux or ANYTHING ELSE; you would have to HOPE you did everything(manually, ew) in the correct order of steps and if you fail then you just completely broke your system and it will not boot again. With NixOS; it does not matter if its KMSCON or Nvidia drivers or Chrome or a desktop envornment or anything, it is always the same easy process.
11. The packages website is objectively what I have always wished for, on any other Linux distro you have to **guess** if you got the name right for a package and if not then too bad.
12. The management of open source and binary packages is unmatched with NixOS. On ArchLinux you would have different package titles for just that, but with NixOS they're the same thing.
13. You make changes entirely from within your IDE. When ricing your system or anything, yo can be writing for hours and when you're done run the rebuild command, and if you didn't like anything then you --rollback. With any other distro you're doing everything by hand which feels like manual labor and SUCKS.
14. Nix-shell is the best linux command period. It changes conceptually how you think about packages, you don't install and uninstall something to use it once, you add it to your shell and when you're done you close the terminal, thats it!
15. It does not matter much that you can reproduce your entire OS from your flake, because you will often never need to do that. But it is useful for if you would like to have it in a VM. but critically although you may never need to do it, it is VITAL that your OS is not tied to one computer, you can rice your OS for WEEKS but if its tied to one computer and you would literally have to set it all up again if something happened.. that's just not great. But with NixOS; you *do not* need to 'create scripts' or 'dotfiles' to backup your system, its built in to how you work!
16. You never have to repeat yourself. I have been using the same flake for years, and I have never once had to redo something because I deleted it and wanted it back, I can just uncomment it or reference the git history for it.
17. On any other Linux OS you would install a thousand packages and forget a thousand more of them. With NixOS your system is only ever exactly what you want or need, not using something?; comment it out!
18. You are never stuck in a hole with NixOS, on any other linux system is possible and in fact common to put yourself in a situation where you HAVE to fix something right now or fix it later but either way; before anything you have to fix something first to be able to use your computer.
* With NixOS; you could have a laggy computer or the best computer state or anything, and at any time if you wanted to go back or undo something; as long as rebuild switch works, you're not stuck. You can instantly switch back to any state you want.
19. If for any reason you have to replace your OS, with NixOS unrelated to home files, you can switch completely to windows or archlinux or linuxmint and then go back to nixos *exactly* how it was in under 30 minutes.
20. Everything is automated, NixOS has the most build in commands of any linux OS and comes with many tools that actually help you and enable you to do your work, with archlinux you would have to install third party untrusted tools and do all the manual labor yourself, with something like rebuild switch; you run one command and wait.
21. NixOS is for every system: your laptop, desktop, server, VMs. There's no such thing as a "gaming" distro, there's distros that support all of the software you want like proton plus and steam without question.
* without nixos, a person might expect "Oh I'll just use Archlinux for my desktop and linuxmint for my laptop and debian for my server and alpine for my vms" which sounds like HELL. Also Debian has **THE MOST** critical software vulnerabilities of any OS objectively.
22. NixOS solves backwards compatability. The Nix Store is an objective end to an era, it is without question the most geinus solution and really concludes the fault. On NixOS not only can you have python3.13 and python 3.14 running at the same time(which I do a lot and is super useful), but the Nix Store supports software of any era all at once, whereas windows is one solid thing and is only upgrading while storing an active copy of its prior versions at all times just so that software will still run.
