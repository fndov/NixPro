Idea: define the profile setting in the build command




How does NixPro work?

set settings in flake
in profile home.nix add imports from user/ and options to the file
in profile configuration.nix add imports from system/ and options to the file

flake.nix
  - SYS ./system/driver/hardware.nix
  - SYS ./system/driver/${settings.system.gpu}.nix
  - SYS ./system/compose/default.nix
  - SYS ./profile/${settings.profile}/configuration.nix
  - SYS ./system/${settings.desktop.type}/default.nix
  - HM  ./profile/${settings.profile}/home.nix

flake.nix
  - SYS ./system/driver/hardware.nix
  - SYS ./system/driver/gpu.nix
  - SYS ./system/compose/default.nix
  - SYS ./profile/profile/configuration.nix
  - SYS ./system/type/default.nix
  - HM  ./profile/profile/home.nix

flake.nix image
  - SYS ./system/driver/hardware.nix
  - SYS ./system/driver/intel.nix
  - SYS ./system/compose/default.nix
  - SYS ./profile/image/configuration.nix
    - SYS ../../system/driver/pipewire.nix
    - SYS ../../system/driver/memory.nix
  - SYS ./system/de/default.nix
  - HM  ./profile/image/home.nix
    - HM ../../user/software/commands/sh.nix        
    - HM ../../user/software/commands/cli.nix       
    - HM ../../user/software/commands/lib.nix
    - HM ../../user/software/apps/terminal/ghostty.nix
    - HM ../../user/software/apps/browser/firefox.nix
